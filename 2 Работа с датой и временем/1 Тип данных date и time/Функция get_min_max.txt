Примечание 1. В тестирующую систему сдайте программу, содержащую только необходимую функцию get_min_max(), но не код, вызывающий ее.

from datetime import date 

def get_min_max(dates):
    if len(dates) == 0:
        return tuple()
    else:
        return (min(dates), max(dates))
